# 📘 Student Marks System — Interactive README

## 🚀 Overview
This is a simple **C# Console Application** that:
- Takes marks input from the user
- Calculates obtained marks
- Calculates percentage
- Displays grade/status based on percentage

---

# 🧠 Features

✅ User Input  
✅ Marks Calculation  
✅ Percentage Calculation  
✅ Grade System  

---

# 📂 Program Flow

```text
Start
   ↓
Enter Subject Marks
   ↓
Calculate Obtained Marks
   ↓
Calculate Percentage
   ↓
Display Grade
   ↓
End
